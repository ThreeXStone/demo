# 退货政策

## 7天无理由退货

- 适用范围：所有数码配件类商品
- 条件：
  1. 收货后7天内
  2. 商品包装完整
  3. 配件齐全（充电线、耳帽、说明书等）
  4. 商品无明显使用痕迹
- 运费：由买家承担（除非商品有质量问题）

## 质量问题退货

- 适用范围：所有商品，无时间限制
- 条件：
  1. 商品存在功能故障
  2. 商品与描述不符
  3. 需提供照片或视频证明
- 运费：由卖家承担

## 不支持退货的情况

- 个人定制商品
- 贴身用品（已拆封）
- 虚拟商品
- 食品、药品（已拆封）

## 退货流程

1. 订单详情页点击"申请售后"
2. 选择"退货退款"
3. 填写退货原因
4. 提交申请，等待审核
5. 审核通过后寄回商品
6. 商家签收后退款
